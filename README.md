<h2 id="english">🇬🇧 English</h2>

Diagoldze is a modern web application built with **Nuxt**. It focuses on delivering a highly interactive and premium user experience by leveraging smooth scrolling and high-performance animations.

### 🚀 Tech Stack

- **Framework**: [Nuxt](https://nuxt.com/) (Vue.js framework)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/) for utility-first responsive styling
- **Animations**: [GSAP](https://gsap.com/) (GreenSock Animation Platform) for complex and performant animations
- **Smooth Scroll**: [Lenis](https://lenis.studiofreight.com/) for a fluid smooth scrolling experience
- **Icons**: [@nuxt/icon](https://nuxt.com/modules/icon)
- **Typography**: [@nuxtjs/google-fonts](https://google-fonts.nuxtjs.org/)
- **Database / Caching**: [ioredis](https://github.com/redis/ioredis)

### 📦 Setup & Installation

Make sure to install the dependencies before running the project:

```bash
# npm
npm install

# pnpm
pnpm install

# yarn
yarn install

# bun
bun install
```

### 💻 Development Server

Start the development server on `http://localhost:3000`:

```bash
# npm
npm run dev

# pnpm
pnpm dev

# yarn
yarn dev

# bun
bun run dev
```

### 🏗️ Production Build

Build the application for production:

```bash
# npm
npm run build

# pnpm
pnpm build
```

Locally preview the production build:

```bash
# npm
npm run preview

# pnpm
pnpm preview
```

<h2 id="bahasa-indonesia">🇮🇩 Bahasa Indonesia</h2>

Diagoldze adalah aplikasi web modern yang dibangun menggunakan framework **Nuxt**. Proyek ini difokuskan pada pengalaman pengguna yang premium dan interaktif dengan memanfaatkan animasi tingkat lanjut dan *smooth scrolling* (gulir halus).

### 🚀 Teknologi yang Digunakan

- **Kerangka Kerja (Framework)**: [Nuxt](https://nuxt.com/) (Framework Vue.js)
- **Desain & Tampilan (Styling)**: [Tailwind CSS](https://tailwindcss.com/) untuk desain yang responsif
- **Animasi**: [GSAP](https://gsap.com/) untuk animasi yang kompleks, mulus, dan berkinerja tinggi
- **Smooth Scroll**: [Lenis](https://lenis.studiofreight.com/) untuk pengalaman *scroll* yang halus dan elegan
- **Ikon**: [@nuxt/icon](https://nuxt.com/modules/icon)
- **Tipografi**: [@nuxtjs/google-fonts](https://google-fonts.nuxtjs.org/)
- **Database / Cache**: [ioredis](https://github.com/redis/ioredis) untuk manajemen data dengan Redis
